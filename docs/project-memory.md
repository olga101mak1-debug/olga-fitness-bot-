## ⚠️ Важно: деплой вручную по SFTP, не через git

- Деплой на сервер идёт вручную по SFTP (paramiko), НЕ через `git pull`.
- Из-за этого правки, сделанные прямо на сервере, могут не попадать обратно в этот GitHub-репозиторий, если их отдельно не закоммитить и не запушить.
- **Подтверждено 23.07.2026:** фикс `max_tokens` (500→2000 в `coach.py`/`weekly_analyst.py`) стоит на сервере, но в этом репозитории всё ещё старое значение — значит, после серверного фикса коммит в GitHub не был сделан.
- Правило на будущее: после любой правки прямо на сервере — обязательно скачать актуальный файл и закоммитить его сюда же, иначе следующая сессия будет работать со устаревшим кодом.
